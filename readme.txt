Getting started with React Native Application Development.
1. run this command to locally package, serve, and publish projects: "npm install expo-cli --global"
2. open the folder using the cd command and run the following command in the folder : "expo start" to run
3. install expo on your iOS/Android device or using android emulator like GenyMotion or iOS emulator in XCODE.
4. run "npm install react-native-elements --save" to install all the components of react native.
